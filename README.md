# Real Estate Market Analysis: CABA & GBA (Argentina)

This project provides a comprehensive end-to-end data pipeline for analyzing the residential rental market in **Buenos Aires (CABA)** and the **Greater Buenos Aires (GBA)**.

## 🚀 Project Overview

The core objective is to transform raw, scraped real estate data into actionable insights through rigorous SQL cleaning and interactive Power BI visualizations.

### 🛠️ Key Components

1.  **Data Extraction (Python):** Raw data was originally scraped from Argenprop, capturing over 10 locations including CABA, San Isidro, Vicente Lopez, and more.
2.  **SQL Data Transformation:**
    *   **Normalization:** Standardized city and neighborhood names, including the assignment of "Comunas" for CABA listings.
    *   **Data Imputation:** Handled missing values (area, expenses, rooms) using **median-based imputation** grouped by geographic clusters.
    *   **Currency Standardization:** Converted all listings to USD using a consistent exchange rate for longitudinal analysis.
    *   **Deduplication:** Identified and removed duplicate listings using CTEs and window functions.
3.  **Power BI Analysis:**
    *   Interactive dashboard showing rental price trends.
    *   Geographic distribution analysis across different "Partidos" and "Comunas".
    *   Key metrics: Average price per m2, expense ratios, and market saturation.

## 📁 Repository Structure

*   `sql/data_cleaning.sql`: The primary SQL script containing the logic for standardization, cleaning, and feature engineering.
*   `powerbi/rent_project.pbix`: The Power BI dashboard file containing the data model and visualizations.
*   `.gitignore`: Configured to exclude heavy raw data files (CSV) and environments.

---
*Note: Raw CSV data files are excluded from this repository due to size and privacy. The cleaning logic provided in the SQL script is designed to run against the schema generated by the initial scrapers.*
